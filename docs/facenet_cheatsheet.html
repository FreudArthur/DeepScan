FaceNet Cheatsheet
Face recognition via embeddings 512-d — vérification, identification, clustering
Comment ça marche
Principe fondamental
Image visage
→
Détection + crop
→
FaceNet CNN
→
Embedding 512-d
→
Distance L2 / cosine
FaceNet ne classifie pas directement. Il projette chaque visage dans un espace métrique où même personne → vecteurs proches, personnes différentes → vecteurs loin.
Triplet Loss (intuition)
# Entraîné avec des triplets (anchor, positive, negative)
# anchor   = visage de référence
# positive = même personne, image différente
# negative = personne différente

# Objectif :
# dist(anchor, positive) + margin < dist(anchor, negative)
# → "serre les même, éloigne les différents"
3 tâches possibles
Vérification 1:1
# "Est-ce la même personne ?"
dist = distance(emb_A, emb_B)
same = dist < threshold
# Seuil typique : 0.6–1.0 (L2)
Identification 1:N
# "Qui est cette personne ?"
# Chercher le + proche dans la DB
dists = [dist(emb_q, e) for e in db]
idx = np.argmin(dists)
# Si dists[idx] < thresh → match
Clustering N:N
# "Regrouper des visages inconnus"
from sklearn.cluster import DBSCAN
labels = DBSCAN(eps=0.5, metric='euclidean').fit_predict(embeddings)
# labels[i] = id du cluster, -1 = outlier
Seuil de distance
Même personne
Personnes différentes
threshold ≈ 0.6–1.0 (L2)  |  ≈ 0.4 (cosine)
Le seuil idéal dépend de ton dataset. Calibre-le sur un val set avec des paires labeled (même / différent).

Initialisation facenet-pytorch
from facenet_pytorch import MTCNN, InceptionResnetV1
import torch

device = 'cuda' if torch.cuda.is_available() else 'cpu'

# Détecteur de visages
mtcnn = MTCNN(
    image_size=160,       # taille output crop
    keep_all=False,       # True → tous les visages
    min_face_size=20,     # ignorer petits visages
    device=device
)

# Modèle d'embedding
resnet = InceptionResnetV1(
    pretrained='vggface2'  # ou 'casia-webface'
).eval().to(device)
VGGFace2 = meilleur en général. CASIA-Webface = plus léger, moins précis sur variations de pose.

Extraire un embedding
from PIL import Image
import torch

img = Image.open("face.jpg")

# Étape 1 : détecter + aligner + crop
face_tensor = mtcnn(img)        # → (3, 160, 160) ou None

if face_tensor is not None:
    face_tensor = face_tensor.unsqueeze(0).to(device)  # (1,3,160,160)

    # Étape 2 : embedding
    with torch.no_grad():
        emb = resnet(face_tensor)      # → (1, 512)
        emb = emb.squeeze()          # → (512,)
        emb = emb.cpu().numpy()       # → np.array (512,)
L'embedding est déjà L2-normalisé par défaut dans InceptionResnetV1. Si tu utilises un autre modèle, normalise manuellement.

Distances
Distance L2
import numpy as np

dist = np.linalg.norm(emb_a - emb_b)
# Seuil recommandé : < 0.6–1.0
same = dist < 0.8
Cosine similarity
from numpy.linalg import norm

cos_sim = np.dot(emb_a, emb_b) / (
    norm(emb_a) * norm(emb_b)
)
# cos_sim ∈ [-1, 1], > 0.9 = même
Vérification 1:1
def verify(img1_path, img2_path, threshold=0.8):
    emb1 = get_embedding(img1_path)
    emb2 = get_embedding(img2_path)
    if emb1 is None or emb2 is None:
        return None
    dist = np.linalg.norm(emb1 - emb2)
    return {"verified": dist < threshold, "distance": float(dist)}

Construire une base d'embeddings enrollment
import numpy as np, pickle
from pathlib import Path

db_embeddings = []
db_labels = []

for person_dir in Path("faces_db/").iterdir():

# Sauvegarder

Reconnaissance (query) 1:N
def recognize(img_path, db_embs, db_labels, threshold=0.8):
    query_emb = get_embedding(img_path)
    if query_emb is None:
        return "no_face"

    # Distances avec toute la DB
    dists = np.linalg.norm(db_embs - query_emb, axis=1)
    idx = np.argmin(dists)

    if dists[idx] < threshold:
        return db_labels[idx], dists[idx]
    return "unknown", dists[idx]
Avec FAISS (grande DB)
FaceNet + FAISS scalable
import faiss

d = 512
index = faiss.IndexFlatIP(d)   # cosine si embeddings normalisés

# Embeddings déjà L2-normalisés → IndexFlatIP = cosine
index.add(db_embeddings.astype('float32'))

# Query
query = query_emb[np.newaxis].astype('float32')
D, I = index.search(query, k=1)

cos_sim = D[0][0]
best_idx = I[0][0]

if cos_sim > 0.6:  # seuil cosine
    print(db_labels[best_idx])
Les embeddings InceptionResnetV1 sont L2-normalisés → IndexFlatIP = cosine similarity directement. Pas besoin de normalize_L2 supplémentaire.

Calibrer le seuil
from sklearn.metrics import roc_curve
import numpy as np

# paires = [(emb_a, emb_b, is_same), ...]
dists = [np.linalg.norm(a - b) for a, b, _ in paires]
labels = [s for _, _, s in paires]

fpr, tpr, thresholds = roc_curve(labels, [-d for d in dists])
# Choisir threshold à FPR = 0.01 (1% faux positifs)
best_thresh = thresholds[np.argmax(tpr[fpr <= 0.01])]

Moyenne d'embeddings (robustesse)
# Enrollment avec plusieurs photos → embedding moyen
# Plus robuste que comparer 1 seule photo

embs_person = [get_embedding(p) for p in photos_of_person]
embs_person = [e for e in embs_person if e is not None]
mean_emb = np.mean(embs_person, axis=0)
